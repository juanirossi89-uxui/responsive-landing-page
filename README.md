# Responsive Landing Page

This project is part of the Responsive Web Design certification from freeCodeCamp.

## Overview
A responsive product landing page built using HTML and CSS, following a mobile-first approach and responsive layout principles.

## Technologies
- HTML5
- CSS3
- Flexbox
- Responsive layouts

## UX Focus
Although this is a front-end project, the main focus was on:
- Clear content hierarchy
- Simple and intuitive navigation
- Readable typography
- Responsive behavior across different screen sizes

## What I learned
- Structuring semantic HTML
- Building layouts with Flexbox
- Applying mobile-first and responsive design principles
- Translating UX concepts into basic front-end implementation
